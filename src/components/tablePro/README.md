# tablePro

##  列宽度的计算

maxWidth：列允许最大宽度，自动分配宽度最大只能为此值

minWidth：列最小允许宽度，自动分配宽度最小只能为此值

width：列的宽度。如果未定义，宽度将自动分配

_width: 表格内部使用的宽度值

宽度计算规则：

1.  在已使用宽度未超出总宽度时

    `剩余宽度` = `总宽度` - `已使用宽度`

    一般情况下，表格不为未设置宽度列自动分配宽度。

    但在 `剩余宽度` 并不足够让未分配宽度的列平摊最小宽度时，需要给未分配宽度的列分配最小宽度。

2.  在已使用宽度超出总宽度时

    在分配的宽度超出屏幕宽度时，需要为未设置屏幕宽度的列分配最小宽度
